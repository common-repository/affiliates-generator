== Description ==

Tool to generate affiliate links quickly and easily.
Allow your affiliates generate links from the frontend.

Just use the shortcode <code>[affiliates-generator]</code>

This plugin requires some version of <a href="http://www.itthinx.com?affiliates=51" target="_blank">Affiliates</a> by @itthinx
<br>Official site: [eggemplo](http://eggemplo.com/plugins/affiliates-generator/)

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip the archive and put the `affiliates-generator` folder into your plugins folder (/wp-content/plugins/).
1. Activate the plugin from the Plugins menu.
